# F1 Race Insights

Small data engineering project exploring Formula 1 race data.

This project pulls race telemetry and lap data to analyze:

- race pace trends
- driver consistency
- pit strategy
- lap time comparisons

Data sources:

FastF1
Ergast F1 API

Example:

python main.py
