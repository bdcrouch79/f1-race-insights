# F1 Race Insights

**A motorsport analytics lab exploring race performance, strategy, and driver behavior through data.**

---

## What This Is

This project analyzes Formula 1 race data to uncover:

- pace trends  
- driver consistency  
- tire degradation  
- race strategy impact  

Built as a hands-on exploration of real telemetry and race data.

---

## Why It’s Interesting

Formula 1 is not just speed.

It’s:

- precision  
- consistency  
- strategy  
- decision-making under pressure  

Small differences in performance create massive outcomes.

This project surfaces those differences.

---

## Data Sources

- FastF1  
- Ergast F1 API
